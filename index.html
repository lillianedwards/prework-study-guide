HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user
  • The title elements represents what is visible on the browser tab.
  • Meta is stored within the head and is the data about the webpage.
  • Headings are larger text lines that organize the webpage.
  • Paragraphs are blocks of test nested under headings.
  • Unordered lists hold list items which contain lists of information relevant to the webpage.
  • Breaks are empty spaces inbetween text or images on webpage.
  • Sections denote where one section or information starts and another begins.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indiactes how much space we want around the content inside of an element.
  • The CSS Box Model consists of an element's borders, padding, and content.
  • The astrik sign apply's a CSS rule to all visible elements on the page.
  • Elements that use the same CSS rule can be seperated by a comma so that the code doesn't have to be repeated.
  • The link to CSS style sheet goes under head and before title in an HTML file.
  • Image changes the properties of an image in an HTML file and can be assigned to all the image elements in the markup.
  • Class attributes are used to add styles to certain elements on the page, a class selector that can be used is card.
  • Hex# are color codes for that can be added into style sheets to broaden the range of colors used on the page.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add: adda file as it looks to next commit
  • git commit -m: commit staged content as new commit snapshot
  • git branch: list of branches appears next to currently active branch
  • git merge: merge the specified branch's history into the current one
  • git clone: retrieve a repository from a online location using a URL
  • git push: adding name of branch transmits branch from local environment to remote repository
  • git pull: fetch and merge any commits from the tracking remote branch

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • The script to a js sheet goes under the footer and above the body elements in an HTML file becuase JS is affecting the HTML therefore it needs to load first.
  • A string variable is a sequence of text that is enclosed in single or double quotation.
  • A number variable is simply a number, not enclosed by anything.
  • A Boolean variable returns a true or false response.
  • An array variable stores multiple values in a single reference. Arrays are indexed starting at 0. Each value goes up one whole number as the amount of values in the array increases.
  • An object variable can be anything, everything in JS is an object and can be stored as a variable.
  • Commenting out code allows it to stay in the file but not be read by the browser, keeping it in case you need it later.
  • Operators determine and show relationships between variables.
  • Conditionals run tests to determine which varibles will run in the event of certain actions and which will not.
  • Functions give pages functionality and need an arguement inside of a parantheses to enact their assigned task.
  • Functions need a string varibale as an argument to tell the function what to display.
  • Events are code structures that listen to activity in the browser and run code in response to that event.
  • JavaScript is a weak typing language meaning variable are not directly associated with any particualr value type and can be assigned and re-assigned to values of all types.
  • For a function to work it needs to be defined first and then called by name.
  • Order matters greatly in JavaScript. The control flow is determined by the order of the code and therefore things need to be placed intentionally so that that right code loads first and can then be changed by precedding codee.
  • The Math function in JavaScript helps generate random numbers and can be used in an array to display a random topic from a list.
